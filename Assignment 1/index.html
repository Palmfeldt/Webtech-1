Startsida och fråga 1

Vad avses med termerna klient och server?

Klienten avser den enhet som hämtar information ifrån en server. Beroende på vilket sammanhang så kan detta vara webbläsaren, operativsystemet eller enheten i säg själv som talas om. Servern delar ut (Hosting) data. En server hostar till exempel webtjänster, filserver, spel etcetera. Alla hemsidor som syns på internet körs på någon form av server.

Beskriv vad var och en av protokollen TCP och IP gör på sändande och mottagande sida.

TCP (Transmission Control Protokoll) är ett dataöverföringsprotokoll på transport lagret i TCP/IP modellen. Protokollet är grunden till andra applikationsprotokoll så som tex http. Protokollet har error checking och skickar om packet (uppdelad data) om de skulle försvinna, ta för långt tid eller bli korrupterade under sändningen. Protokollet använder sig av en den så kallade ”three way handshake” som skickar SYN, SYN ACK och ACK som responskoder. Detta signalerar för klienten och server när en koppling har lyckats göras. IP (Internet protocol) är ett kommunikationsprotokoll som ligger på internet lagret i TCP/IP modellen. Den hanterar skickandet av packet från klienten till server (samt vice versa, eller flera enheter samtidigt)
