Vad är en regel i CSS och vad består den av?

En regel i CSS är en gruppering av en eller flera egenskaper som kan sedan läggas till för en viss sorts element i HTML. T.ex. div {

Color…

}

Vad är skillnaden mellan id och class? När bör man använda id respektive class?

Ett Id används endast för ett enda element (dock så valideras detta ej i webbläsaren) medan en klass kan användas på flera valda element. Id kan i så fall användas för enstaka speciella element medans classer kan användas på flera olika element som måste se lika ut. Om du assignar ett id till t.ex. en header så kan du sedan specificera samma id i URL adressen för att direkt komma till den. Te.x. wikipedia.org/wiki/Lenovo#Historik
