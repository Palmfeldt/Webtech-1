Startsida och fråga 1

Fråga 1

  • Då man skapar en layout med grid används bl.a. CSS-egenskaperna grid-template-columns, grid-template-rows och grid-template-areas. Beskriv vad dessa används till.

Grid template (colums, rows) specificerar antal och storleken på rows och colums i en grid layout d.v.s. display:grid. De kan specificeras med values så som length men även max-content (som utökar sig till maxstorleken till innehållet den anpassar) etcetera. Areas definierar var och vad som finnas i rutnätet (samt ger ”cellerna” namn).

Vad är en media query och vad används den till?

Media queries är hur en hemsida kan anpassa sig till olika enheter och dess skärmar. Detta görs i CSS och det kan kolla brädden och längden av skärmar, resolutionen, skärmorientering (d.v.s. portrait eller landscape mode) etcetera. I detta fall finns det tre sätt som denna hemsida anpassar sig till.
